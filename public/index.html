HTML

HTML es un lenguaje de etiquetas para el desarrollo estructural de las paginas web siendo lo mas vital en una pagina y a su vez lo mas sencillo, una mejor explicacion es pensar que una pagina web es un cuerpo el sistema oseo seria HTML ya que es la escrutura que sostiene todo, JavaScript seia los organos y mulculos ya que son el funcionamiento principal de el cuerpo y con el cual puede hacer tareas y CSS seria la apareciancia de cuarpo como la piel ya que es el aspecto que forma
